# Weather App

### Overview

This is a simple weather application that allows users to check the current weather and a 5-day forecast for a specific city. Users can also fetch weather data based on their current location.

### Features

- Search weather by city name

- Get current location weather

- View temperature, humidity, and wind speed

- Display a 5-day weather forecast

- Store recent searches for quick access

## Technologies Used

- HTML

- CSS (Tailwind CSS)

- JavaScript

- OpenWeatherMap API

### How to Use

- Enter a city name in the input field and click the "Search" button.

- Click "Use Current Location" to get weather data for your current location.

- View recent searches in the dropdown menu.

- Weather details and a 5-day forecast will be displayed upon successful API response.

## Installation & Setup

- Clone the repository:

```
git clone https://github.com/your-repo/weather-app.git
```

- Navigate to the project folder:

```
cd weather-app
```

Open `index.html` in a browser.

### API Key

Replace API_KEY in script.js with your own OpenWeatherMap API key:

```
const API_KEY = "your_api_key_here";
```
